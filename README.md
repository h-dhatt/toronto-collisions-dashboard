# toronto-collisions-dashboard

**Goal:** Analyze collision trends by time and police division in Toronto.
**Tools:** SQL, Power BI (Service)

## What I built
- Monthly collision trend (A)
- Collisions by hour (B)
- Top divisions by collisions (C)
- Fatalities by division (D)

## Key insights
- Highest collision division: **NSC**
- Highest fatalities division: **D42**
- Collision volume varies by hour and month (see dashboard)

## Data
Toronto Traffic Collisions Open Data (downloaded as CSV).
